[搭建博客心得]

July 24, 2017 本文总阅读量次 696

搭建这个博客用到的方法和遇到坑的解决过程

我是linux搭建的，
1.下载GIT
点击进入廖雪峰的GIT教程，查看安装部分并按教程安装就OK了

2.下载安装Node.js
后文的npm开头的命令都需要依赖Node.js。直接到Node官网下载安装，看到最大的那两个绿色按钮，下载v6的就行。下载下来以后一路next下一步就可以了。

3.下载安装hexo
HEXO官网说得很详细，包括如何部署在GITHUB上面，不太懂可以查看 这篇文章

4.适配主题
推荐使用Next 主题,，既简约又功能多还很完善

5.绑定个性域名

  • 阿里云
    国内域名绑定首选，坑比较少，不过要身份验证什么的注册后就要等多几天，而且我注册完就没有.me域名了，心塞，我就换为GOdaddy了
    教程
  • GOdaddy
    有中文，支持支付宝支付，除了阿里云外就选他了，不过坑有点多。。。
    教程

6.遇到的坑。。。

  • 输入hexo g 后显示 hexo 命令找不到
    把npm install hexo-cli -g换为npm install hexo -g就行了

  • npm下载总卡主
    换为淘宝镜像就解决了 教程

  • GOdaddy绑定域名
    Markdown
    关注类型为A和类型为CNAME名称为WWW这两个就好了
    比如我的博客域名为 so1n.me
    A的值 为自己再电脑ping (库名).github.io 后得到的值也可以用github提供的
    设置好后（CNAME文件也设置好）访问so1n.me后就会跳转为 （库名）github.io
    www的值为 (库名).github.io 设置后，访问www.so1n.me 后就会跳转为so1n.me
